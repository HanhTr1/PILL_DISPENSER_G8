# PILL_DISPENSER_G8_PROJECT
Automated pill dispenser with LED signal, stepper motor control, optical piezo sensor, LoRaWAN communication and stored in non-volatile memory EEPROM
## MININUM REQUIREMENTS
#### BUTTON & LED HANDLING
#### STEPPER MOTOR CONTROL
#### OPTICAL SENSOR (CALIBRATION)
#### PIEZO SENSOR (PILL DETECTION)
## ADVANCED REQUIREMENTS
#### EEPROM
#### LORAWAN COMMUNICATION
#### DETECT POWER STATUS
#### RECALIBRATE
## STATE MACHINE
 - Wait for the button
 - Calibration
 - Ready to run
 - Dispensing (30s)
 - Finished (7 turns)
 - Error
